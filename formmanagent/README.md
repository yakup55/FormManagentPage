# Form Yönetim Sayfası Projesi

Bu projenin yaparken backend alanında Asp.NetCore Web Api kullandım .Fronted alanında React.js(redux) javascript kütüphanesini kullandım.Tasarım için bootsrap kullandım

## Kullanılan Teknolojiler

**İstemci:** React, Redux

**Sunucu:** .Net Core

## Kullanıcı Neler Yapabilir Neler Görebilir

- Kullanıcılar tüm form yazılarını görebilir,formlarda arama yapabilir
- Kullanıcı sisteme üye olabilir
- Kullanıcı kendi sayfasında form ekleyebilir eklediği formlarda arama yapabilir

## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/yakup55/FormManagentPage.git
```

Proje dizinine gidin

```bash
  cd hotelreservation
```

Gerekli paketleri yükleyin

```bash
  npm install
```

Sunucuyu çalıştırın

```bash
  npm run start
```
